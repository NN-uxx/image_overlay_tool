# ツール名前
オ―バ―レイ バッチツール

## 簡単な説明
二種類のpng形式の画像ファイルを重ねて合成します。
上から判子を押していくのをイメージすると分かりやすいです。

## 必要要件
Windows11

## 使用ライブラリー
tkinter
PIL

## 使い方
1・適当なフォルダに合成したい画像ファイルを入れてください。入れたファイルが下地になります。（面倒なら同梱のinputフォルダに入れてください）
2・上から重ねたい画像ファイルを任意の場所に用意してください。これが判子になります。（サンプルとしてOverlay.pngが入っています）
3・OverlayTool.pyを起動します
4・入力フォルダに1のフォルダを選択してください
5・レイヤー画像に2の画像ファイルを選択してください
6・バッチ処理開始をボタンをクリックしてください
7・1で指定したフォルダの下にoutputフォルダが作成され、その中に合成されたファイルが作成されます

## 注意事項
・透過画像を扱う都合、対応している画像形式はpngファイルのみです。
・判子ファイルの位置は自動で右下となります。
・マシンパワーにも拠りますが、少し処理が重いです。
　重さは枚数や画像ファイルの大きさに比例するので10枚程度から試してみてください。

## インストール
解凍するだけです。アンインストールはフォルダごと消してください
